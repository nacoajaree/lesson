        <h2>3. สื่อกลางในการส่งข้อมูล (Transmission Media)</h2>

        <p>สื่อกลางเป็นทางผ่านของสัญญาณข้อมูล แบ่งออกเป็น 2 ประเภทใหญ่ๆ ดังนี้:</p>

        

        <h3>1. สื่อกลางแบบมีสาย (Guided Media)</h3>

        <ul>

          <li><strong>สายคู่บิดเกลียว (Twisted Pair):</strong> ที่เราคุ้นเคยในชื่อ "สายแลน (LAN)" มีทั้งแบบไม่มีฟอยล์หุ้ม (UTP) และมีฟอยล์หุ้ม (STP) เพื่อลดสัญญาณรบกวน</li>

          <li><strong>สายแกนร่วม (Coaxial):</strong> สายทองแดงเส้นเดี่ยวที่มีฉนวนหนา มักใช้กับสายสัญญาณเคเบิลทีวี หรือสายกล้องวงจรปิดรุ่นเก่า</li>

          <li><strong>สายใยแก้วนำแสง (Fiber Optic):</strong> ทำจากใยแก้วหรือพลาสติก ส่งข้อมูลโดยใช้ <u>สัญญาณแสง</u> มีความเร็วสูงมาก ระยะทางไกล และไม่ถูกรบกวนด้วยคลื่นแม่เหล็กไฟฟ้า</li>

        </ul>


        <h3>2. สื่อกลางแบบไร้สาย (Unguided Media)</h3>

        <ul>

          <li><strong>คลื่นวิทยุ (Radio Wave):</strong> ใช้ในการสื่อสารทั่วไป เช่น Wi-Fi, Bluetooth ส่งสัญญาณไปได้ทุกทิศทาง</li>

          <li><strong>ไมโครเวฟ (Microwave):</strong> ส่งสัญญาณเป็นเส้นตรงระดับสายตา (Line-of-Sight) มักใช้เชื่อมต่อระหว่างสถานีฐานหรือตึกสูง</li>

          <li><strong>ดาวเทียม (Satellite):</strong> สถานีไมโครเวฟลอยฟ้า รับสัญญาณจากพื้นโลก (Uplink) และส่งกลับลงมา (Downlink) ครอบคลุมพื้นที่กว้างขวางระดับโลก</li>

        </ul>

        <h2>6. Client-Server Architecture, อินเทอร์เน็ต และ IP Address</h2>

        

        <h3>สถาปัตยกรรมแบบผู้รับบริการและผู้ให้บริการ (Client-Server)</h3>

        <p>เป็นรูปแบบที่คอมพิวเตอร์ <strong>Client (เครื่องลูกข่าย)</strong> ทำการร้องขอข้อมูล (Request) ผ่านเครือข่ายไปยัง <strong>Server (เครื่องแม่ข่าย)</strong> ซึ่งเป็นเครื่องคอมพิวเตอร์ประสิทธิภาพสูงที่เปิดรอให้บริการตลอดเวลา เมื่อ Server ได้รับคำร้องขอ ก็จะประมวลผลแล้วส่งข้อมูลตอบกลับ (Response)</p>

        

        <h3>IP Address & DNS</h3>

        <ul>

          <li><strong>IP Address:</strong> หมายเลขประจำเครื่องคอมพิวเตอร์บนระบบเครือข่าย (เช่น 192.168.1.1 หรือ IPv6 รุ่นใหม่) เพื่อไม่ให้ส่งข้อมูลผิดเครื่อง</li>

          <li><strong>DNS (Domain Name System):</strong> เนื่องจากตัวเลข IP จำยาก มนุษย์จึงตั้งระบบแปลงชื่อเว็บ (เช่น google.com) ให้กลายเป็นหมายเลข IP Address ที่คอมพิวเตอร์เข้าใจ</li>

        </ul>


        <h3>ความเป็นมาของอินเทอร์เน็ตในประเทศไทย</h3>

        <p>อินเทอร์เน็ตเริ่มต้นจากโครงการ **ARPANET** ของกระทรวงกลาโหมสหรัฐฯ ส่วนในประเทศไทย เริ่มต้นขึ้นในปี **พ.ศ. 2530** โดยสถาบันเทคโนโลยีแห่งเอเชีย (AIT) และมหาวิทยาลัยสงขลานครินทร์ ได้ทดลองเชื่อมต่ออีเมลกับมหาวิทยาลัยในออสเตรเลีย ต่อมาเครือข่ายคอมพิวเตอร์เชิงพาณิชย์รายแรกจึงเริ่มเกิดขึ้นและแพร่หลายในเวลาต่อมา</p>

  // ระบบสลับเนื้อหาบทเรียน

  function switchSection(index) {

    if(index < 0 || index >= totalSections) return;

    

    // ซ่อนหน้าปัจจุบันและเปิดหน้าใหม่

    document.getElementById(`sec-${currentSection}`).classList.remove('active');

    document.getElementById(`sec-${index}`).classList.add('active');

    

    // อัปเดตสถานะเมนูด้านข้าง

    const navItems = document.querySelectorAll('.nav-item');

    navItems[currentSection].classList.remove('active');

    navItems[index].classList.add('active');

    

    currentSection = index;

    visitedSections.add(index);

    

    // จัดการเปิด/ปิด ปุ่ม ย้อนกลับ/ถัดไป

    document.getElementById('btnPrev').disabled = (currentSection === 0);

    if(currentSection === totalSections - 1) {

      document.getElementById('btnNext').style.display = 'none';

      checkCompletionStatus();

    } else {

      document.getElementById('btnNext').style.display = 'block';

    }

    

    updateProgress();

    window.scrollTo({top: 0, behavior: 'smooth'});

  }


  function moveSection(step) {

    switchSection(currentSection + step);

  }


  // คำนวณความคืบหน้า %

    document.getElementById('progressBar').style.width = pct + '%';

  }


  // ตรวจสอบสถานะการทำครบเงื่อนไข

      resStatus.innerText = `เข้าเรียนครบแต่ยังทำข้อสอบไม่ครบ (ทำแล้ว ${totalQuizAnswered}/6 ข้อ)`;

    }

  }


  // ฟังก์ชันระบบตรวจข้อสอบ Interactive

  function checkQuiz(quizId, correctAnswer) {

    const selected = document.querySelector(`input[name="q${quizId}"]:checked`);

    const fb = document.getElementById(`fb-${quizId}`);

    

    if(!selected) {

    }
