1. การสื่อสารข้อมูลเบื้องต้น (Data Communication)

การสื่อสารข้อมูล คือ กระบวนการถ่ายโอนหรือแลกเปลี่ยนข้อมูลกันระหว่างผู้ส่งและผู้รับ ผ่านช่องทางสื่อกลาง โดยใช้กฎเกณฑ์หรือข้อตกลงร่วมกัน เพื่อให้เกิดความเข้าใจตรงกัน

องค์ประกอบ 5 ส่วนที่สำคัญที่สุด

*คลิกองค์ประกอบแต่ละส่วนเพื่อศึกษาความหมาย

1. ผู้ส่ง (Sender) 🖱️ คลิกเพื่อดูรายละเอียด
2. ผู้รับ (Receiver) 🖥️ คลิกเพื่อดูรายละเอียด
3. ข่าวสาร (Message) ✉️ คลิกเพื่อดูรายละเอียด
4. สื่อกลาง (Media) 🔌 คลิกเพื่อดูรายละเอียด
5. โปรโตคอล (Protocol) 📜 คลิกเพื่อดูรายละเอียด

3. สื่อกลางในการส่งข้อมูล (Transmission Media)

สื่อกลางเป็นทางผ่านของสัญญาณข้อมูล แบ่งออกเป็น 2 ประเภทใหญ่ๆ ดังนี้:

1. สื่อกลางแบบมีสาย (Guided Media)

  • สายคู่บิดเกลียว (Twisted Pair): ที่เราคุ้นเคยในชื่อ "สายแลน (LAN)" มีทั้งแบบไม่มีฟอยล์หุ้ม (UTP) และมีฟอยล์หุ้ม (STP) เพื่อลดสัญญาณรบกวน
  • สายแกนร่วม (Coaxial): สายทองแดงเส้นเดี่ยวที่มีฉนวนหนา มักใช้กับสายสัญญาณเคเบิลทีวี หรือสายกล้องวงจรปิดรุ่นเก่า
  • สายใยแก้วนำแสง (Fiber Optic): ทำจากใยแก้วหรือพลาสติก ส่งข้อมูลโดยใช้ สัญญาณแสง มีความเร็วสูงมาก ระยะทางไกล และไม่ถูกรบกวนด้วยคลื่นแม่เหล็กไฟฟ้า

2. สื่อกลางแบบไร้สาย (Unguided Media)

  • คลื่นวิทยุ (Radio Wave): ใช้ในการสื่อสารทั่วไป เช่น Wi-Fi, Bluetooth ส่งสัญญาณไปได้ทุกทิศทาง
  • ไมโครเวฟ (Microwave): ส่งสัญญาณเป็นเส้นตรงระดับสายตา (Line-of-Sight) มักใช้เชื่อมต่อระหว่างสถานีฐานหรือตึกสูง
  • ดาวเทียม (Satellite): สถานีไมโครเวฟลอยฟ้า รับสัญญาณจากพื้นโลก (Uplink) และส่งกลับลงมา (Downlink) ครอบคลุมพื้นที่กว้างขวางระดับโลก
📝 Quick Check: ข้อใดคือเหตุผลสำคัญที่สุดที่ระบบอินเทอร์เน็ตความเร็วสูงตามบ้านในปัจจุบัน เปลี่ยนมาใช้สาย Fiber Optic แทนสายโทรศัพท์ทองแดงแบบเดิม?

4. เครือข่ายคอมพิวเตอร์ และโครงสร้างรูปแบบ (Topology)

เครือข่ายคอมพิวเตอร์ (Computer Network) คือ กลุ่มของอุปกรณ์คอมพิวเตอร์ที่นำมาเชื่อมต่อกันเพื่อให้เกิดการใช้ทรัพยากรร่วมกัน เช่น การแชร์ไฟล์ การใช้เครื่องพิมพ์ร่วมกัน หรือการต่ออินเทอร์เน็ต

รูปแบบการเชื่อมต่อโครงสร้างเครือข่าย (Network Topology)

แบบบัส (Bus Topology) 🚌 ใชเกนสายสัญญาณหลักร่วมกัน (Bus) สัญญาณวิ่งผ่านสายหลัก หากสายหลักขาด เครือข่ายทั้งหมดจะล่มทันที
แบบดาว (Star Topology) ⭐ อุปกรณ์ทุกตัวเชื่อมต่อเข้ากับ "อุปกรณ์ศูนย์กลาง" (เช่น Switch) นิยมมากที่สุดในปัจจุบัน เพราะหากสายใดสายหนึ่งขาด เครื่องอื่นยังทำงานได้ปกติ
แบบวงแหวน (Ring Topology) ⭕ ส่งข้อมูลวนไปในทิศทางเดียวกันเป็นวงกลม เครื่องที่ถือตั๋วพิเศษ (Token) เท่านั้นถึงจะส่งได้ หากเครื่องใดพัง เครือข่ายจะขาดตอน
แบบตาข่าย (Mesh Topology) 🕸️ อุปกรณ์ทุกเครื่องมีสายต่อเชื่อมไปยังเครื่องอื่นๆ โดยตรงทุกเครื่อง มีความทนทานต่อความผิดพลาดสูงสุด (Fault Tolerance) แต่สิ้นเปลืองค่าสายส่งมาก
📝 Quick Check: ในห้องแล็บคอมพิวเตอร์แห่งหนึ่ง หากสายแลนของเครื่องหมายเลข 5 ขาดเสียหาย แต่เครื่องอื่นๆ ในห้องยังคงใช้งานอินเทอร์เน็ตได้ตามปกติ แสดงว่าห้องแล็บนี้เชื่อมต่อเครือข่ายแบบใด?

5. อุปกรณ์ระบบเครือข่าย (Network Devices)

ในการทำให้คอมพิวเตอร์ติดต่อกันได้ จำเป็นต้องมีอุปกรณ์เฉพาะทางทำหน้าที่จัดการสัญญาณและเส้นทาง

การ์ดเครือข่าย (NIC / LAN Card) แผงวงจรในคอมพิวเตอร์ที่ทำหน้าที่แปลงข้อมูลดิจิทัลให้กลายเป็นสัญญาณส่งออกไป และมีที่อยู่ทางกายภาพที่แก้ไขไม่ได้เรียกว่า MAC Address
สวิตช์ (Switch) อุปกรณ์เชื่อมสายแลนในเครือข่ายเดียวกัน (LAN) มีความฉลาดเพราะจะส่งข้อมูลไปหา เฉพาะพอร์ตที่เป็นปลายทางเท่านั้น โดยอ้างอิงจาก MAC Address
เราเตอร์ (Router) อุปกรณ์ที่ทำหน้าที่เชื่อมต่อเครือข่ายที่ อยู่คนละวงเครือข่าย เข้าด้วยกัน (เช่น เครือข่ายในบ้านเชื่อมกับอินเทอร์เน็ตโลกภายนอก) มีหน้าที่เลือกเส้นทางที่ดีที่สุดในการส่งข้อมูลโดยอ้างอิงจาก IP Address
แอกเซสพอยต์ (Access Point - AP) ทำหน้าที่กระจายสัญญาณไวไฟ (Wi-Fi) แปลงสัญญาณดิจิทัลจากสายแลนให้กลายเป็นคลื่นวิทยุกระจายไปในอากาศ
📝 Quick Check: หากต้องการเชื่อมโยงเครือข่ายภายในบ้านออกไปสู่อินเทอร์เน็ตของค่ายผู้ให้บริการ (ISP) เพื่อเข้าเว็บไซต์ต่างประเทศ จำเป็นต้องใช้อุปกรณ์ข้อใดทำหน้าที่ค้นหาและเลือกเส้นทางข้ามเครือข่าย?

6. Client-Server Architecture, อินเทอร์เน็ต และ IP Address

สถาปัตยกรรมแบบผู้รับบริการและผู้ให้บริการ (Client-Server)

เป็นรูปแบบที่คอมพิวเตอร์ Client (เครื่องลูกข่าย) ทำการร้องขอข้อมูล (Request) ผ่านเครือข่ายไปยัง Server (เครื่องแม่ข่าย) ซึ่งเป็นเครื่องคอมพิวเตอร์ประสิทธิภาพสูงที่เปิดรอให้บริการตลอดเวลา เมื่อ Server ได้รับคำร้องขอ ก็จะประมวลผลแล้วส่งข้อมูลตอบกลับ (Response)

IP Address & DNS

  • IP Address: หมายเลขประจำเครื่องคอมพิวเตอร์บนระบบเครือข่าย (เช่น 192.168.1.1 หรือ IPv6 รุ่นใหม่) เพื่อไม่ให้ส่งข้อมูลผิดเครื่อง
  • DNS (Domain Name System): เนื่องจากตัวเลข IP จำยาก มนุษย์จึงตั้งระบบแปลงชื่อเว็บ (เช่น google.com) ให้กลายเป็นหมายเลข IP Address ที่คอมพิวเตอร์เข้าใจ

ความเป็นมาของอินเทอร์เน็ตในประเทศไทย

อินเทอร์เน็ตเริ่มต้นจากโครงการ **ARPANET** ของกระทรวงกลาโหมสหรัฐฯ ส่วนในประเทศไทย เริ่มต้นขึ้นในปี **พ.ศ. 2530** โดยสถาบันเทคโนโลยีแห่งเอเชีย (AIT) และมหาวิทยาลัยสงขลานครินทร์ ได้ทดลองเชื่อมต่ออีเมลกับมหาวิทยาลัยในออสเตรเลีย ต่อมาเครือข่ายคอมพิวเตอร์เชิงพาณิชย์รายแรกจึงเริ่มเกิดขึ้นและแพร่หลายในเวลาต่อมา
